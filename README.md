# ShiftGuard

**A Human-in-the-Loop Framework for Distribution Shift Detection, Attribution, and Adaptive Retraining in Non-Stationary Forex Markets**

ShiftGuard is an ML pipeline that detects, explains, and adapts to distribution shifts in forex markets. It combines event-aware shift detection, SHAP-based attribution, and a human-in-the-loop Streamlit dashboard for model oversight and retraining.
